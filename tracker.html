☁️ Back up to Google Drive (optional, one-time setup)

This saves an encrypted-in-transit copy of your data to a private file in your own Google Drive — only this app can read or write that file, and nothing is shared with anyone else, including us. It requires a free Google OAuth Client ID that only you control:

  1. Go to Google Cloud Console → Credentials and create a project (any name).
  2. Enable the Google Drive API for that project (search for it under "APIs & Services").
  3. Configure the OAuth consent screen as "External", add your own Gmail address as a test user, and keep it in "Testing" mode — no Google review needed for personal use.
  4. Create an OAuth Client ID of type "Web application". Under "Authorized JavaScript origins", add the exact URL this page is hosted at (e.g. https://yourusername.github.io).
  5. Copy the Client ID (ends in .apps.googleusercontent.com) and paste it below.

Nothing here is secret — a Client ID on its own can't access your data without you approving the sign-in popup each time.
